C++的奇妙知识

  • main函数的返回值必须为int(应该是根据编译器的不同而定的)。
  • C++程序根据编译器的不同，有不同的后缀名，常见的包括.cc、.cxx、.cpp、.cp及.C。
  • echo命令可以获取main函数的返回值 
    $ echo $? // UNIX
$ echo %ERRORLEVEL% // Windows
  • 术语“流”(Steam)表达的是，随着时间推移，字符串是顺序生成或消耗的。
